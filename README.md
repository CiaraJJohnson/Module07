# Module12

## Description:

-This project focuses on applying business analytics techniques to analyze sales, customer behavior, and operational performance for a fictional grocery company called GreenGrocer. It also demonstrates the use of data analysis, visualization, and basic predictive modeling to generate business valuable insights and recommendations.

## Files Included:

module12_assignment.py – Main Python script containing all data generation, analysis, and visualizations

## What I Practiced:

-Using pandas and NumPy for data manipulation
-Creating data visualizations with Matplotlib
-Performing descriptive and diagnostic analyticsn
-Generating business insights and recommendations

## Key Features:
-Simulated real-world business data (sales, customers, stores)
-Sales performance analysis by store and department
-Executive summary with actionable recommendations
Summary

This project demonstrates how data analytics can be used to support business decision-making by identifying trends, understanding customer behavior, and forecasting future performance.
